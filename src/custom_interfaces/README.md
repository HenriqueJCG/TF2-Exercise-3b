#custom_interfaces package

Package to illustrate the ROS 2 definition of a custom interface.

Used in the [Communications using topics tutorial](https://sir.upc.edu/projects/ros2tutorials/2-communications_using_topics/index.html) of the Introduction to ROS course.

Related info:
- [Tutorial Creating custom msg and srv files](https://docs.ros.org/en/humble/Tutorials/Beginner-Client-Libraries/Custom-ROS2-Interfaces.html)
- [Tutorial Implementing custom interfaces](https://docs.ros.org/en/humble/Tutorials/Beginner-Client-Libraries/Single-Package-Define-And-Use-Interface.html)

## Usage

This package is used in package *pubsub_custom* where a publisher and a subscriber comunicate using the messages described here.
